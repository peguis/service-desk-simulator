# INC-002 - Computador Lento

## Identificação

ID: INC-002

Data: 19/08/2026

Usuário: Usuário Fictício

Analista: Pedro Guilherme

Status: Resolvido

## Informações do Chamado

Categoria: Hardware/Desempenho

Tipo: Incidente

Título: Computador apresentando lentidão

Descrição:

Usuário relatou que o computador está demorando para iniciar e abrir aplicativos, impactando suas atividades diárias.

## Classificação

Impacto: Baixo

Urgência: Média

Prioridade: P3

## Diagnóstico

Sintomas identificados:

Inicialização lenta do sistema
Demora para abrir programas
Travamentos ocasionais

Testes realizados:

Verificação do uso de CPU
Verificação do consumo de memória RAM
Verificação do uso do disco
Análise de programas configurados para iniciar com o Windows

Resultado:

Foram identificados diversos programas iniciando automaticamente com o sistema, consumindo recursos desnecessários.

## Ações Realizadas
1. Acessado o Gerenciador de Tarefas.
2. Analisados os programas de inicialização.
3. Desativados aplicativos não essenciais na inicialização automática.
4. Reiniciado o computador para validação.

## Solução

Após a remoção de programas desnecessários da inicialização, o sistema apresentou melhora significativa no desempenho e redução do tempo de inicialização.
