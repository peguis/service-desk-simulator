# Base de Conhecimento - Windows


## Problema

Computador apresenta lentidão ou baixo desempenho.


---

## Sintomas

- Windows inicia lentamente
- Programas demoram para abrir
- Sistema trava
- Alto consumo de recursos


---

## Possíveis causas

- Muitos programas iniciando com o Windows
- Alto uso de CPU
- Falta de memória RAM
- Pouco espaço em disco
- Processos em segundo plano


---

## Diagnóstico

Verificar:

Gerenciador de Tarefas:

- Uso de CPU
- Memória RAM
- Disco
- Aplicativos em execução


Verificar inicialização:

- Programas iniciando automaticamente


---

## Solução

Realizar:

- Fechar programas desnecessários
- Desativar aplicativos da inicialização
- Liberar espaço em disco
- Reiniciar computador


---

## Boas práticas

- Manter Windows atualizado
- Remover programas sem uso
- Evitar instalar softwares desconhecidos


---

## Escalonamento

Encaminhar para N2 caso:

- Suspeita de falha de hardware
- Problema no disco
- Necessidade de upgrade
