# Base de Conhecimento - Senha Bloqueada


## Problema

Usuário não consegue acessar um sistema devido ao bloqueio da conta.


---

## Sintomas

- Login recusado
- Mensagem de usuário bloqueado
- Falha de autenticação


---

## Possíveis causas

- Muitas tentativas de senha incorreta
- Senha expirada
- Problema de sincronização


---

## Diagnóstico

Verificar:

- Nome do usuário
- Status da conta
- Permissões de acesso


---

## Solução

Realizar:

- Desbloqueio da conta
- Redefinição de senha
- Teste de login


---

## Boas práticas

Orientar usuário:

- Não compartilhar senha
- Usar senhas fortes
- Ativar MFA


---

## Escalonamento

Encaminhar para N2 caso:

- Problema persista após redefinição
- Erro de permissão
- Problema no servidor de autenticação
